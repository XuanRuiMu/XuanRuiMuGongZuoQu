# 完整验证清单

当需要完整31项检查清单、逐项判定标准或阈值说明时使用本文件。

## A. 占位符残留（7项检查）

| #   | 检查项      | 模式     | 阈值   | 失败处理              |
| --- | ----------- | -------- | ------ | --------------------- |
| A1  | ×号         | \u00d7   | 零容忍 | 定位段落，在run中替换 |
| A2  | 小写xxxx    | "xxxx"   | 零容忍 | 替换                  |
| A3  | 大写XXXX    | "XXXX"   | 零容忍 | 替换                  |
| A4  | 小写xxx     | "xxx"    | 零容忍 | 替换                  |
| A5  | 大写XXX     | "XXX"    | 零容忍 | 替换                  |
| A6  | 白色方块    | \u25a1   | 零容忍 | 通常是\u3000→空格修复 |
| A7  | 4个全角空格 | \u3000×4 | 零容忍 | 替换为普通空格        |

例外：×在数学上下文中是合法的（乘法、系数、公式引用）。如果段落包含乘法、系数、式（等关键词则跳过。目录条目（_Style
28/TOC样式）中的×占位符也跳过，因为目录由用户在Word中手动更新。

## B. 封面字段完整性（8项检查）

| #   | 检查项       | 验证方式                         |
| --- | ------------ | -------------------------------- |
| B1  | 标题非空     | 文档文本中包含"RPG"或"我的世界"  |
| B2  | 学生姓名     | 存在姓名字符串（如"魏建国"）     |
| B3  | 学号         | 存在学号（如"6022203183"）       |
| B4  | 指导教师     | 存在教师姓名字符串（如"陈宏江"） |
| B5  | 学校名称     | 存在学校名称字符串               |
| B6  | 院系         | 存在院系字符串                   |
| C7  | 专业         | 存在专业字符串                   |
| B8  | 封面标题标题 | 存在"毕业设计"或"论文"           |

注意：这些检查使用全文档文本的子串匹配。具体值取决于用户的实际数据。

## C. 结构完整性（6项检查）

| #   | 检查项         | 阈值                                 | 备注         |
| --- | -------------- | ------------------------------------ | ------------ |
| C1  | 中文摘要存在   | 前80段中含"摘"+"要"                  | 检查摘要标题 |
| C2  | 英文摘要存在   | 前100段中含"ABSTRACT"                | 不区分大小写 |
| C3  | 章节结构       | 文本中含"第"且含"章"字符             | 中文章节编号 |
| C4  | 参考文献部分   | 至少出现一次"参考文献"或"References" |              |
| C5  | 表格存在       | len(doc.tables) > 0                  | 报告数量     |
| C6  | 标题样式已使用 | 标题+标题样式段落计数                | 报告计数     |

## D. 格式一致性（5项检查）

| #   | 检查项      | 阈值                   | 备注                   |
| --- | ----------- | ---------------------- | ---------------------- |
| D1  | 段落数合理  | 50 < 数量 < 2000       | 太少=不完整；太多=臃肿 |
| D2  | Run结构保留 | 平均runs/para > 1.5    | 低平均值=格式丢失      |
| D3  | 下划线存在  | 带下划线的非空runs > 0 | 签名行应有下划线       |
| D4  | 无方框字符  | 任何段落中无\u25a1     | Unicode伪影检查        |
| D5  | 标题中无×   | 标题样式段落中无\u00d7 | 封面标题清洁           |

## E. 内容质量（5项检查）

| #   | 检查项       | 阈值                    | 备注                                             |
| --- | ------------ | ----------------------- | ------------------------------------------------ |
| E1  | 空段落       | < 80                    | 大量空段落暗示格式碎片（模版本身含约73个空段落） |
| E2  | 无超大段落   | 无超过2000字符的段落    | 超大=可能粘贴错误                                |
| E3  | 总字数       | 5000-200000字符         | 超出范围=异常                                    |
| E4  | 中文字符充足 | > 1000个中文字符        | 论文必须以中文为主                               |
| E5  | 无重度重复   | 无50字符片段重复超过3次 | 复制粘贴检测                                     |

总计：**31项检查**。全部通过方可提交。
