## 1. 基础乐理速查

### 音程表

| 音程              | 半音数 | 示例 | 和声色彩       | 旋律感觉           |
| ----------------- | ------ | ---- | -------------- | ------------------ |
| **纯一度**        | 0      | C-C  | 完全融合       | 静止               |
| **小二度**        | 1      | C-Db | 紧张、碰撞     | 上行紧迫/下行解决  |
| **大二度**        | 2      | C-D  | 中性、民族风   | 自然级进           |
| **小三度**        | 3      | C-Eb | 忧伤、柔和     | 小调特征音程       |
| **大三度**        | 4      | C-E  | 明亮、稳定     | 大调特征音程       |
| **纯四度**        | 5      | C-F  | 开放、支撑     | 上行有力/下行温和  |
| **增四度/减五度** | 6      | C-F# | 极度紧张、不安 | "魔鬼音程"，需解决 |
| **纯五度**        | 6      | C-G  | 稳定、空灵     | 强力进行           |
| **小六度**        | 8      | C-Ab | 悲剧性、渴望   | 大调中的"暗色"     |
| **大六度**        | 9      | C-A  | 温暖、宽广     | 向上开放           |
| **小七度**        | 10     | C-Bb | 属七和弦特征   | 下行解决倾向       |
| **大七度**        | 11     | C-B  | 现代、张力     | 需要解决到八度     |
| **纯八度**        | 12     | C-C' | 强化、统一     | 跳跃或强调         |

### 三和弦类型

```python
chord_types = {
    "大三和弦 (Major)": {
        "结构": "根音 + 大三度 + 纯五度",
        "示例": "C-E-G",
        "色彩": "明亮、稳定、快乐",
        "符号": "C, CMaj, I",
        "功能": "主功能和弦（最稳定）",
    },

    "小三和弦 (Minor)": {
        "结构": "根音 + 小三度 + 纯五度",
        "示例": "C-Eb-G",
        "色彩": "忧郁、柔和、紧张",
        "符号": "Cm, Cmin, i",
        "功能": "主功能（小调）",
    },

    "增三和弦 (Augmented)": {
        "结构": "根音 + 大三度 + 增五度",
        "示例": "C-E-G#",
        "色彩": "不稳定、漂浮、神秘",
        "符号": "Caug, C+, I+",
        "功能": "过渡和弦，需要解决",
    },

    "减三和弦 (Diminished)": {
        "结构": "根音 + 小三度 + 减五度",
        "示例": "C-Eb-Gb",
        "色彩": "极度紧张、收缩",
        "符号": "Cdim, C°, i°",
        "功能": "通常作为属七和弦的一部分",
    },
}

```

### 七和弦类型

```python
seventh_chords = {
    "属七和弦 (Dominant 7th)": {
        "结构": "大三和弦 + 小七度",
        "示例": "G-B-D-F",
        "色彩": "强烈需要解决到主和弦",
        "功能": "V7 → I 最重要进行",
        "常用转位": "原位, 第一转位(6/5), 第二转位(4/3), 第三转位(4/2)",
    },

    "大七和弦 (Major 7th)": {
        "结构": "大三和弦 + 大七度",
        "示例": "C-E-G-B",
        "色彩": "梦幻、爵士、现代",
        "功能": "IMaj7, IVMaj7（大调中）",
    },

    "小七和弦 (Minor 7th)": {
        "结构": "小三和弦 + 小七度",
        "示例": "C-Eb-G-Bb",
        "色彩": "柔和、爵士、灵魂乐",
        "功能": "ii7, iii7, vi7（大调）；i7, iv7（小调）",
    },

    "半减七和弦 (Half-diminished 7th)": {
        "结构": "减三和弦 + 小七度",
        "示例": "C-Eb-Gb-Bb",
        "色彩": "复杂、微妙张力",
        "符号": "Cm7b5, ø7",
        "功能": "viiø7（导七和弦）",
    },

    "减七和弦 (Fully Diminished 7th)": {
        "结构": "减三和弦 + 减七度",
        "示例": "C-Eb-Gb-Bbb(A)",
        "色彩": "极度紧张、黑暗",
        "功能": "vii°7（等音转换可能）",
    },
}

```

## 2. 调性与调号

### 大小调关系

```text
         大调 ←→ 关系小调
    ┌─────────────────────┐
C大调  ←→   a小调 (0个升降)
G大调  ←→   e小调 (1升)
D大调  ←→   b小调 (2升)
A大调  ←→   #f小调 (3升)
E大调  ←→   #c小调 (4升)
B大调  ←→   #g小调 (5升)
#F大调 ←→   #d小调 (6升)
#C大调 ←→   #a小调 (7升)

F大调  ←→   d小调 (1降)
Bb大调 ←→   g小调 (2降)
Eb大调 ←→   c小调 (3降)
Ab大调 ←→   f小调 (4降)
Db大调 ←→   bb小调 (5降)
Gb大调 ←→   eb小调 (6降)
Cb大调 ←→   ab小调 (7降)

```

### 调性色彩

| 调性       | 情绪联想           | 典型用法                     |
| ---------- | ------------------ | ---------------------------- |
| **C大调**  | 纯粹、简单、童真   | 教育、民谣、儿童音乐         |
| **G大调**  | 明亮、田园、民间   | 乡村、民谣、赞美诗           |
| **D大调**  | 英雄、胜利、庄严   | 进行曲、fanfare、颂歌        |
| **A大调**  | 浪漫、温暖、抒情   | 爱情主题、慢板乐章           |
| **Eb大调** | 宏伟、史诗、崇高   | 交响乐、宗教音乐、英雄主义   |
| **Bb大调** | 欢快、友好、朴素   | 庆典、舞曲、室内乐           |
| **F大调**  | 田园、牧歌、宁静   | 自然场景、冥想               |
| **d小调**  | 悲剧、命运、斗争   | 交响乐第一乐章、戏剧音乐     |
| **e小调**  | 忧郁、沉思、内省   | 夜曲、浪漫曲、悲歌           |
| **c小调**  | 悲壮、革命、激情   | 贝多芬式英雄主义、葬礼进行曲 |
| **#f小调** | 热情、激烈、痛苦   | 浪漫主义高潮段落             |
| **g小调**  | 特殊、异域、悲剧性 | 巴洛克风格、西班牙风味       |

## 3. 常用和声进行

### 功能性和声进行

#### 大调中的基本进行

```python
major_progressions = {
    # === 完全进行 (Complete Cadence) ===
    "完全终止": "I → IV → V7 → I",
                 # 最稳定的结束感

    "变格终止": "I → IV → I",
                # 教堂风格，宁静

    "阻碍终止": "V7 → vi",
               # 未完成的期待，继续发展

    # === 常用扩展进行 ===
    "流行进行": "I → V → vi → IV (循环)",
              # 无数流行歌曲的基础

    "圆圈进行": "vi → ii → V → I (或 I→V→vi→iii→IV→I→IV→V)",
              # 古典到现代都适用

    "Pachelbel进行": "I → V → vi → iii → IV → I → IV → V",
                     # Canon in D 的基础

    # === 特殊效果进行 ===
    "欺诈进行": "V → VI (大调中)",
              # 意外的转折

    "那不勒斯进行": "i → bII6 → V7 → I",
                   # 极度悲伤/戏剧性
}

```

#### 小调中的基本进行

```python
minor_progressions = {
    "自然小调": "i → VII → III → VI → II → V(V7) → i",

    "和声小调": "i → iv → V7 → i (升高VII级)",
             # 最常用的小调和声

    "旋律小调上行": "i → IV → V7 → i (升高VI/VII)",
                  "旋律小调下行": "i → VII → VI → v → i (还原VI/VII)",

    "小调特有进行": [
        "i → VI (常见，明亮化)",
        "iv → VI (弗里吉亚进行)",
        "VII → i (导音到主音)",
    ],
}

```

### 和弦外音 (Non-harmonic Tones)

```python
non_harmonic_tones = {
    "经过音 (Passing Tone)": {
        "定义": "在两个不同和弦音之间级进的音",
        "记谱": "P 或无标记",
        "示例": "C - D - E (D是经过音)",
        "效果": "流畅的线条连接",
    },

    "辅助音 (Neighbor Tone)": {
        "定义": "离开和弦音再回到同一音（上行或下行）",
        "记谱": "N 或无标记",
        "示例": "E - F - E (F是辅助音)",
        "效果": "装饰、延迟解决",
    },

    "倚音 (Appoggiatura)": {
        "定义": "强拍上的非和弦音，跳进后级进解决",
        "记谱": "App.",
        "示例": "F - E - D (F是倚音，跳进到E)",
        "效果": "情感强调、表现力强",
    },

    "延留音 (Suspension)": {
        "定义": "前一个和弦的音延续到下一个和弦（不协和）后再解决",
        "记谱": "Sus. (Sus2/Sus4)",
        "示例": "G(属) → G-C( sus4) → G-F(解决)",
        "效果": "期待感、拉长紧张",
    },

    "先现音 (Anticipation)": {
        "定义": "提前出现下一和弦的和弦音",
        "记谱": "Ant.",
        "效果": "连接、预示",
    },

    "避留音 (Escape Tone)": {
        "定义": "离开和弦音级进到一个音再跳回另一个和弦音",
        "效果": "装饰性",
    },
}

```

## 5. 旋律写作原则

### 好旋律的特征

```python
good_melody_criteria = {
    "轮廓清晰": "有明确的起伏线（波浪形、上升、下降、拱形）",

    "平衡性": {
        "上行与下行": "大致平衡（除非有意偏向一方）",
        "跳进与级进": "混合使用（跳进创造兴趣，级进保持流畅）",
        "长音与短音": "交替（避免单调）",
    },

    "目标导向": "有方向感（向某个高点或终点前进）",

    "可记忆性": {
        "重复": "适度重复关键动机",
        "对比": "有变化的重复（不完全相同）",
        "高潮点": "有一个或几个明显的高潮音",
    },

    "节奏活力": "有节奏趣味（切分、附点、休止等）",

    "音域合理": "适合演奏者/演唱者的舒适范围",
}

```

### 旋律发展技巧

```python
melody_development = {
    "重复 (Repetition)": "完全重复或稍作改变",

    "模进 (Sequence)": "在同一高度模式上移动",
    "例: C-D-E → D-E-F → E-F#-G (上行模进)",

    " inversion (倒影)": "上下翻转音程方向",
    "例: C-E-G → C-Ab-F (原: 上行大三度+小三度 → 后: 下行)",

    "逆行 (Retrograde): "从后往前演奏",
    "例: C-D-E-F → F-E-D-C",

    "扩大/缩小 (Augmentation/Diminution)":
        "扩大: 所有时值×2",
        "缩小: 所有时值÷2",

    "截断 (Truncation)": "取主题的前半或后半部分",

    "分裂 (Fragmentation)": "把主题拆成小块重新组合",

    "扩展 (Extension)": "在主题末尾添加新材料",

    "对位 (Counterpoint)": "同时演奏原始主题和新线条",
}

```

### 不同风格的旋律特征

| 风格          | 典型特征           | 音域                 | 节奏               | 和声暗示       |
| ------------- | ------------------ | -------------------- | ------------------ | -------------- |
| **古典时期**  | 平衡、对称、动机式 | 中等（一个八度左右） | 规律、方整         | 清晰的功能性   |
| **浪漫时期**  | 长线条、情感丰富   | 较宽（可达两个八度） | 自由rubato         | 复杂、半音化   |
| **民族风格**  | 五声音阶、特殊调式 | 可窄可宽             | 特殊节奏型         | 简单或特色和声 |
| **现代电影**  | 易记、简洁、强烈   | 适中                 | 规律或有明显节奏点 | 功能性强       |
| **电子/流行** | 循环性、hook导向   | 较窄（便于传唱）     | 强节奏驱动         | 简单循环       |

## 6. 节奏与节拍

### 常用拍号

```python
time_signatures = {
    "4/4拍 (Common Time)": {
        "特点": "最通用，强弱次强",
        "适用": "几乎所有风格",
        "细分": "4个四分音符 = 8个八分音符 = 16个十六分音符",
    },

    "3/4拍 (Waltz Time)": {
        "特点": "强-弱-弱，舞蹈感",
        "适用": "圆舞曲、抒情段落",
    },

    "6/8拍 (Compound Duple)": {
        "特点": "两组三个八分音符，摇摆感",
        "适用": "抒情、流动的音乐",
    },

    "2/2拍 (Cut Time)": {
        "特点": "两拍子，快速的感觉",
        "适用": "进行曲、快板乐章",
    },

    "5/4拍 (Quintuple)": {
        "特点": "不对称，2+3或3+2分组",
        "适用": "现代、爵士、实验性",
        "名作": "Take Five (Paul Desmond/Dave Brubeck)",
    },

    "变化拍号": {
        "用法": "在一首作品中切换不同拍号",
        "效果": "增加趣味、配合内容变化",
        "注意": "需要平滑过渡（通过共同细分单位）",
    }
}

```
