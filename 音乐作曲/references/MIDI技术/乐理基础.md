## 音阶与调性

### 大调音程

```text
全 - 全 - 半 - 全 - 全 - 全 - 半
(全 = 全音, 半 = 半音)
```

### 常见调及其音符

| 调     | 音符             |
| ------ | ---------------- |
| C大调  | C D E F G A B    |
| G大调  | G A B C D E F#   |
| D大调  | D E F# G A B C#  |
| A大调  | A B C# D E F# G# |
| F大调  | F G A B♭ C D E   |
| B♭大调 | B♭ C D E♭ F G A  |

### 小调音阶（自然小调）

```text
全 - 半 - 全 - 全 - 半 - 全 - 全
```

| 调    | 音符           |
| ----- | -------------- |
| A小调 | A B C D E F G  |
| E小调 | E F# G A B C D |
| D小调 | D E F G A B♭ C |

## 和弦构造

### 三和弦（3个音）

| 类型     | 公式        | 示例（C）   |
| -------- | ----------- | ----------- |
| 大三和弦 | 1 - 3 - 5   | C - E - G   |
| 小三和弦 | 1 - ♭3 - 5  | C - E♭ - G  |
| 减三和弦 | 1 - ♭3 - ♭5 | C - E♭ - G♭ |
| 增三和弦 | 1 - 3 - #5  | C - E - G#  |

### 七和弦（4个音）

| 类型 | 公式              | 示例（C）       |
| ---- | ----------------- | --------------- |
| 大七 | 1 - 3 - 5 - 7     | C - E - G - B   |
| 属七 | 1 - 3 - 5 - ♭7    | C - E - G - B♭  |
| 小七 | 1 - ♭3 - 5 - ♭7   | C - E♭ - G - B♭ |
| 减七 | 1 - ♭3 - ♭5 - ♭♭7 | C - E♭ - G♭ - A |

### 大调中的自然和弦

| 级数 | 和弦类型 | C大调中 |
| ---- | -------- | ------- |
| I    | 大三     | C       |
| ii   | 小三     | Dm      |
| iii  | 小三     | Em      |
| IV   | 大三     | F       |
| V    | 大三     | G       |
| vi   | 小三     | Am      |
| vii° | 减三     | B°      |

## 终止式（乐句结尾）

| 终止         | 进行     | 感觉         | 用途          |
| ------------ | -------- | ------------ | ------------- |
| **正格终止** | V → I    | 强解决       | 作品/段落结尾 |
| **半终止**   | 任意 → V | 不完整、期待 | 乐句中间停顿  |
| **变格终止** | IV → I   | 柔和、"阿门" | 温柔结尾      |
| **阻碍终止** | V → vi   | 意外         | 延长乐句      |

### C大调示例

```text
正格终止:  G → C   (强结束)
半终止:    C → G   (停顿，期待更多)
变格终止:  F → C   (柔和结束)
阻碍终止:  G → Am  (意外，继续)
```

## 安全音符组合

### 协和音程（听起来稳定）

- 同度、八度、纯五度、纯四度、大三度、小三度、大六度、小六度

### 不协和音程（谨慎使用）

- 小二度（1半音）- **避免**
- 大二度（2半音）- 仅作经过音
- 三全音（6半音）- 快速解决
- 大七度（11半音）- 仅爵士

## 避免同时演奏的音符

| 音符1 | 避免与 |
| ----- | ------ |
| C     | C#、B  |
| D     | D#、C# |
| E     | F、D#  |
| F     | F#、E  |
| G     | G#、F# |
| A     | A#、G# |
| B     | C、A#  |

**规则**：避免相距1个半音的音符同时发声。

## 推荐和弦声部排列

**钢琴/键盘**：将音符分散到不同八度

```text
左手:  C2 - G2
右手:  E4 - G4 - C5
```

**合奏**：将和弦音分配给不同乐器

```text
低音:     C2 (根音)
弦乐:     E3, G3 (三音, 五音)
钢琴:     C4, E4, G4 (完整和弦)
```
