# 声部进行参考

---

## 核心原则

### 1. 平滑进行

每个声部以尽可能小的音程移动。

**正确：**

```text
声部1: C4 → B3 → C4  (级进)
声部2: E4 → D4 → E4  (级进)
声部3: G4 → G4 → G4  (共同音)
```

**错误：**

```text
声部1: C4 → G4 → C5  (大跳)
声部2: E4 → B3 → G4  (不规则)
```

### 2. 反向进行

尽可能让声部朝相反方向移动。

### 3. 避免平行五度和八度

永远不要让两个声部以平行纯五度或纯八度移动。

## 音程规则

### 自由使用的音程

| 音程   | 半音数 | 性质             |
| ------ | ------ | ---------------- |
| 同度   | 0      | 完全协和         |
| 小三度 | 3      | 不完全协和       |
| 大三度 | 4      | 不完全协和       |
| 纯四度 | 5      | 协和（上下文中） |
| 纯五度 | 7      | 完全协和         |
| 小六度 | 8      | 不完全协和       |
| 大六度 | 9      | 不完全协和       |
| 八度   | 12     | 完全协和         |

### 需避免或谨慎使用的音程

| 音程   | 半音数 | 规则                  |
| ------ | ------ | --------------------- |
| 小二度 | 1      | **避免** - 刺耳不协和 |
| 大二度 | 2      | 仅作经过音，不要持续  |
| 三全音 | 6      | 立即解决              |
| 大七度 | 11     | 仅爵士，下行解决      |

## 常见错误

### 1. 音簇（相邻音符）

多个音符在1-2个半音内同时发声。解决方案：将音符分散到不同八度。

### 2. 未解决的不协和音

三全音或七度悬而不决。解决方案：解决到协和音。

### 3. 声部交叉

较低声部高于较高声部。解决方案：保持声部顺序。

### 4. 加倍导音

两个声部在第7级音上。解决方案：改为加倍根音或五音。

## 四部写作指南

### 声部间距

| 声部      | 最大间距  |
| --------- | --------- |
| 女高-女中 | 八度      |
| 女中-男高 | 八度      |
| 男高-男低 | 八度+五度 |

### 音域建议

| 声部/乐器    | 舒适音域 |
| ------------ | -------- |
| 男低         | C2 - E4  |
| 大提琴/男高  | C3 - G4  |
| 中提琴/女中  | C3 - C5  |
| 小提琴/女高  | G3 - E6  |
| 钢琴（左手） | C2 - C4  |
| 钢琴（右手） | C4 - C6  |

## 多轨编配技巧

### 避免频率遮蔽

两个乐器在同一八度演奏相同音符会导致频率遮蔽。解决方案：分散到不同八度。

### 低音+和弦关系

| 和弦  | 首选低音 | 避免        |
| ----- | -------- | ----------- |
| C大调 | C或G     | E（弱）     |
| Am    | A或E     | C（转位感） |
| Dm7   | D或A     | F（不稳定） |

---

## MIDI生成前快速检查

- [ ] 无1个半音内的音符同时发声
- [ ] 低音演奏和弦根音或五音
- [ ] 轨道间无平行五度或八度
- [ ] 不协和音程解决到协和音
- [ ] 声部保持在舒适音域
- [ ] 轨道分散在不同八度
