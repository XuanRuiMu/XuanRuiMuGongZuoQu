# 和弦进行参考

---

## 通用进行（适用于任何风格）

### I - V - vi - IV（"流行"进行）

```text
C大调: C - G - Am - F
G大调: G - D - Em - C
D大调: D - A - Bm - G
```

**用途**：流行、摇滚、叙事曲、乡村

### I - IV - V - I（"布鲁斯"进行）

```text
C大调: C - F - G - C
G大调: G - C - D - G
```

**用途**：布鲁斯、摇滚、民谣

### ii - V - I（"爵士"进行）

```text
C大调: Dm7 - G7 - Cmaj7
F大调: Gm7 - C7 - Fmaj7
```

**用途**：爵士、波萨诺瓦、精致流行

### I - vi - IV - V（"50年代"进行）

```text
C大调: C - Am - F - G
G大调: G - Em - C - D
```

**用途**：嘟·喔普、叙事曲、怀旧感

## 风格特定进行

### 古典/巴洛克

```text
I - IV - V - I       (基本终止)
I - V - vi - iii - IV - I - IV - V  (卡农)
i - VII - VI - V     (安达卢西亚终止，小调)
```

### 爵士

```text
Imaj7 - vi7 - ii7 - V7         (回旋)
Imaj7 - IV7 - iii7 - VI7       (节奏变化A段)
ii7 - V7 - Imaj7 - VI7         (标准回旋)
```

### 电子/氛围

```text
I - III - IV - iv    (调式混合)
i - VII - VI - VII   (史诗/电影)
I - V - vi - iii     (振奋恍惚)
```

### 悲伤/忧郁

```text
i - VI - III - VII   (小调史诗)
i - iv - V - i       (小调终止)
vi - IV - I - V      (情感流行)
```

## 低音线模式

### 根音运动

演奏每个和弦的根音。

```text
和弦: C - G - Am - F
低音: C2  G2  A2   F2
```

### 根音-五音模式

在根音和五音之间交替。

```text
和弦: C
低音: C2 - G2 - C2 - G2
```

### 行走贝斯（爵士）

用级进运动连接和弦。

```text
和弦: Dm7 - G7 - Cmaj7
低音: D2 - E2 - F2 - F#2 | G2 - A2 - B2 - C3 | C3...
```

## 调性关系

### 五度圈（用于转调）

```text
C → G → D → A → E → B → F# → C#
↑                              ↓
F ← B♭ ← E♭ ← A♭ ← D♭ ← G♭ ←
```

### 关系小/大调

| 大调 | 关系小调 |
| ---- | -------- |
| C    | Am       |
| G    | Em       |
| D    | Bm       |
| F    | Dm       |
| A    | F#m      |
| E    | C#m      |

## 和弦替代

| 原和弦 | 替代 | 效果               |
| ------ | ---- | ------------------ |
| V      | vii° | 更柔和的解决       |
| IV     | ii   | 更精致             |
| I      | vi   | 阻碍终止           |
| V7     | ♭II7 | 三全音替代（爵士） |
