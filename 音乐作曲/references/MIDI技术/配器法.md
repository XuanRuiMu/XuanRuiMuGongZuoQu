## 乐器组合

### 加倍（同音）

| 组合        | 效果         |
| ----------- | ------------ |
| 小提琴+长笛 | 明亮、清晰   |
| 大提琴+大管 | 深沉、丰富   |
| 小号+小提琴 | 英雄旋律     |
| 弦乐+铜管   | 史诗、饱满   |
| 钢琴+弦乐   | 温暖、电影感 |

### 分层（不同音）

```text
示例：C大调和弦

长笛:    G5 (五音, 高)
小提琴:  E4 (三音, 中)
中提琴:  C4 (根音, 中)
大提琴:  G3 (五音, 低)
低音:     C2 (根音, 基础)
```

## 动态分层

### 构建强度

| 级别      | 乐器            | 力度  |
| --------- | --------------- | ----- |
| 1（安静） | 独奏钢琴或弦乐  | pp-p  |
| 2（柔和） | 弦乐+木管       | p-mp  |
| 3（中等） | 全弦乐+轻铜管   | mp-mf |
| 4（响亮） | 全管弦乐        | mf-f  |
| 5（高潮） | 全管弦乐+打击乐 | f-ff  |

### 渐强构建

```text
小节1-2: 仅弦乐 (p)
小节3-4: +木管 (mp)
小节5-6: +圆号 (mf)
小节7-8: +小号+定音鼓 (f)
```

## 风格特定编配

### 古典/管弦乐

```text
旋律:     小提琴I / 长笛
和声:     小提琴II / 中提琴 / 圆号
低音:     大提琴 / 低音提琴
强调:     定音鼓 / 铜管短促音
```

### 爵士组合

```text
旋律:     萨克斯 / 小号
和声:     钢琴（和弦声部）
低音:     立贝（行走贝斯）
节奏:     鼓 / 钢琴伴奏
```

### 摇滚乐队

```text
旋律:     人声 / 主音吉他
和声:     节奏吉他 / 键盘
低音:     电贝斯
节奏:     鼓
```

### 电影配乐（史诗）

```text
基础:     低弦乐+铜管
中层:     全弦乐+圆号
顶层:     合唱+高弦乐+小号
节奏:     太鼓+定音鼓
```

## 频率考量

### 避免频率冲突

| 音域            | 需分离的乐器                 |
| --------------- | ---------------------------- |
| 低频（C1-C3）   | 贝斯、大号、大提琴、钢琴左手 |
| 中低频（C3-C4） | 次中音萨克斯、长号、中提琴   |
| 中频（C4-C5）   | 人声、吉他、钢琴、小提琴     |
| 高频（C5-C7）   | 长笛、短笛、高弦乐           |

### 按乐器角色的EQ空间

```text
低音:     50-250 Hz  (基础)
节奏:     250-500 Hz (主体)
和声:     500-2000 Hz (温暖)
旋律:     2000-8000 Hz (存在感)
空气感:   8000+ Hz (明亮)
```

### 最小编配（3-4轨）

```json
{
  "tracks": [
    { "instrument": "acoustic-grand-piano", "role": "旋律+和声" },
    { "instrument": "acoustic-bass", "role": "低音" },
    { "instrument": "string-ensemble-1", "role": "铺底" }
```

## 快速检查清单

- [ ] 每件乐器在其舒适音域内
- [ ] 低音和旋律清晰分离（2+八度）
- [ ] 同八度内无频率冲突
- [ ] 力度适合风格
- [ ] 乐器选择匹配风格
- [ ] 主奏乐器清晰可闻
