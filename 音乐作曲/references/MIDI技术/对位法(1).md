# 对位法参考

对位法是两个或多个同时发声的音乐声部之间的关系，它们在和声上相互依存但在旋律上相互独立。

---

## 分类对位法概述

基于约翰·约瑟夫·福克斯的《通往帕尔纳索斯的阶梯》（1725年），分类对位法通过五个递进复杂度级别教授作曲。

### 协和音分类

| 类型       | 音程                                           |
| ---------- | ---------------------------------------------- |
| 完全协和   | 同度、纯五度、纯八度                           |
| 不完全协和 | 小三度、大三度、小六度、大六度                 |
| 不协和     | 小二度、大二度、纯四度、三全音、小七度、大七度 |

---

## 第一类（音符对音符）

对位声部的每个音符对应固定旋律的一个音符。

### 规则

1. **开始于**：同度、纯五度或纯八度
2. **结束于**：同度或纯八度
3. **仅使用协和音程**：P1、m3、M3、P5、m6、M6、P8
4. **禁止**：平行五度和平行八度

### 进行类型

| 进行 | 描述                     | 评价            |
| ---- | ------------------------ | --------------- |
| 反向 | 声部朝相反方向移动       | ✓ 首选          |
| 斜向 | 一个声部保持，另一个移动 | ✓ 良好          |
| 同向 | 同方向，不同音程         | 谨慎使用        |
| 平行 | 同方向，同音程           | ✗ 避免纯五/纯八 |

---

## 第二类（2:1）

固定旋律每个全音符对应对位声部的两个音符。

### 第二类规则

1. **强拍**：必须协和
2. **弱拍**：可使用经过音（不协和）
3. **相邻强拍无平行纯五/纯八**
4. **以反向或斜向进行接近完全协和**

---

## 第三类（4:1）

固定旋律每个音符对应四个音符。

### 第三类规则

1. **第1拍**：必须协和
2. **第2-4拍**：可使用经过音/邻音
3. **经过音模式**：主音→离开→返回
4. **同方向不超过3个音符**

---

## 第四类（延留）

切分节奏通过延留创造张力和解决。

### 延留结构

| 阶段 | 描述               | 拍位 |
| ---- | ------------------ | ---- |
| 准备 | 协和               | 弱拍 |
| 延留 | 不协和（持续过来） | 强拍 |
| 解决 | 协和（下行级进）   | 弱拍 |

### 常见延留

| 类型 | 音程模式    | 解决     |
| ---- | ----------- | -------- |
| 4-3  | P4→M3或m3   | 下行级进 |
| 7-6  | m7/M7→m6/M6 | 下行级进 |
| 9-8  | M9/m9→P8    | 下行级进 |

---

## 第五类（华彩对位）

结合之前所有类别，使用变化的节奏。

### 指南

1. **混合节奏**：全音符、二分音符、四分音符
2. **不单一类别**占主导
3. **包含延留**以增加和声趣味
4. **保持旋律流畅**，以级进为主

### 对位法检查清单

- [ ] 无平行五度或八度
- [ ] 强拍协和
- [ ] 声部主要以级进移动
- [ ] 优先使用反向进行
- [ ] 不协和音正确准备和解决
- [ ] 旋律线可唱/可奏

## 巴赫风格考量

巴赫的对位法与严格分类不同：

- 自由使用完全和不完全协和音
- 从和声背景生长
- 声部间大胆独立
- 包含更多半音进行

对于巴赫风格赋格，考虑添加：

- 主题（主旋律）
- 答题（移调回应）
- 对题（伴随旋律）
- 插段（发展性段落）
