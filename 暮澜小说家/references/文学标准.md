# 文学标准参考

此文件定义暮澜纪元小说的文风规范、禁忌和文学参照系映射。

- 克里斯蒂·戈登《上古之战》三部曲（魔兽世界小说）——基础文风
- 刘慈欣《三体》——宏大叙事与冷峻笔触

## 文学参照系映射表

**核心原则：任何场景的创作或修改，都必须有明确的文学参照系。不同场景类型需要不同的参照系。**

| 场景类型          | 推荐参照系                            | 核心技法                                                                       |
| ----------------- | ------------------------------------- | ------------------------------------------------------------------------------ |
| 政治会议/权力博弈 | 希拉里·曼特尔《狼厅》三部曲           | 亲密第三人称、展示而非告诉、身体即权力、旁观者沉默即参与、空间即压迫、灰度人物 |
| 战争/大规模战斗   | 托尔金《魔戒》圣盔谷/帕兰诺           | 多线叙事交织、战场感官沉浸、个体命运与宏大战争的对位                           |
| 单兵/小队战斗     | 克里斯蒂·戈登《上古之战》             | 动作即性格、战斗中的情感闪回、武器作为角色延伸                                 |
| 人物独白/内心挣扎 | 陀思妥耶夫斯基《罪与罚》              | 意识流片段、自我对话、道德困境的层层递进                                       |
| 阴谋/暗线推进     | 约翰·勒卡雷《锅匠，裁缝，士兵，间谍》 | 信息不对称叙事、信任的瓦解、沉默比台词更危险                                   |
| 角色关系/情感     | 加西亚·马尔克斯《霍乱时期的爱情》     | 时间跨度中的情感变化、细节作为情感的载体                                       |
| 末日/灾难         | 刘慈欣《三体》黑暗森林                | 冷峻的宇宙观、文明尺度的恐惧、个体的渺小与尊严                                 |
| 日常/生活场景     | 契诃夫短篇小说                        | 留白、潜台词、日常中的裂隙                                                     |

## 文风核心原则

### 冷峻、扎实、电影化

- 通过具体动作传达情感：不写"他很悲伤"，写"他的手指在弓弦上停了一瞬，然后松开了"
- 通过表情传达内心：不写"她很愤怒"，写"她的下颌收紧，目光从对方脸上滑过，落在桌上的弩箭上"
- 通过环境映射心境：不写"气氛紧张"，写"营地的火堆噼啪作响，没有人说话。末影珍珠在袋中微微震颤"

### 展示而非告诉（曼特尔《狼厅》核心技法）

**这是最容易违反的原则，必须反复检查。**

| 告诉（❌）       | 展示（✅）                       | 区别                                         |
| ---------------- | -------------------------------- | -------------------------------------------- |
| "嘴唇在发抖"     | "嘴闭得很紧"                     | 前者直接说"恐惧"，后者可以是愤怒也可以是恐惧 |
| "没有人看他"     | "凡迪安在翻报表。提奥在看天花板" | 前者直接说"孤立"，后者让读者自己发现         |
| "他知道下一句话" | "水面轻轻颤了一下"               | 前者是内心独白，后者是物理细节               |
| "不自觉地往后靠" | "背贴上了椅背"                   | "不自觉地"是告诉，"贴上"暗示身体自己退了     |
| "呼吸声变得粗重" | "呼吸声填满了那段沉默"           | 前者用形容词标签，后者让读者自己感受         |
| "脊背僵直了"     | "背挺直了"                       | "僵直"是被吓住，"挺直"可以是准备战斗         |

### 展示不是比喻概括（关键修正）

**展示而非告诉只解决了"告诉"的问题，没有解决"用比喻概括情感"的问题。** 精妙比喻依然是概括，不是展示。

| AI化展示（❌）                                   | 人类化展示（✅）                                   | 区别                                               |
| ------------------------------------------------ | -------------------------------------------------- | -------------------------------------------------- |
| "脑子里有什么碎了"                               | 他反复检查弩箭的装填，明明已经装好了               | 前者用比喻概括恐惧，后者用不合理的行为展现恐惧     |
| "像一台过载的机器自动关闭了某些非必要的功能"     | 他站在那里，过了好一会儿才想起来自己应该跑         | 前者用技术比喻概括麻木，后者用行动的延迟展现麻木   |
| "一种更安静的、更沉的东西，像一块被水浸透的石头" | 他蹲下来，把散落的石子一颗一颗捡回筐里，筐早就烂了 | 前者用比喻概括沉重，后者用无意义的重复动作展现沉重 |
| "恐惧需要未知，而她现在面对的不是未知"           | 她盯着那盏灯看了很久，然后伸手把它拧暗了一格       | 前者是哲学分析，后者是一个说不清理由的动作         |

**核心原则**：真正的展示是写角色做了什么不合逻辑的事，让读者自己感受到情感。比喻再精妙，也是从外部观察情感；行动的错位，才是从内部经历情感。

### 对话规范

- 真人说话是碎片化的：允许不完整句子、打断、口语
- 方言角色保持方言：露利萨的天津味必须贯穿始终
- 暴雪经典台词可引用：但需自然融入，不生硬
- 不同角色语音必须可区分：遮住名字也能认出是谁在说话

### 叙事节奏

- 场景推进：每个场景必须推进情节、深化角色、或让世界更真实，否则删除
- 闲笔不是浪费，是呼吸——至少10%的内容不推动情节、不点明主题，只让世界更真实
- 信息控制：读者需要的信息通过动作和对话自然释放，不通过旁白灌输
- 留白：关键情感时刻用沉默和动作，不用大段内心独白
- 前松后紧：铺陈阶段气定神闲，转折阶段急转直下（曼特尔《提堂》节奏）
- 温度波动：叙事温度必须随场景自然升降，不允许全文恒温——冷峻是基调，不是恒温箱。恐怖前可以有日常的温暖，绝望中可以有荒诞的幽默，紧张中可以有士兵式的黑色幽默

### 身体即权力指示器（曼特尔《狼厅》核心技法）

- 权力关系通过身体传达，不是通过形容词
- 谁站着、谁坐着、谁靠近门、谁背对窗，都是权力关系
- 被压迫者的身体反应形成递进线，每一步比上一步更接近崩溃
- 旁观者的沉默不是中立，而是选择——描述每个人在做什么，让读者自己发现回避

### 多感官描写规范

**核心问题**：AI写作的感官描写几乎只有视觉，偶尔有标签化的嗅觉/听觉说明。人类体验世界是全感官的——闭上眼睛也能感受到场景。

**自检方法**：写完一段后闭上眼睛重读。如果闭上眼睛就感受不到这个场景，说明只有视觉，补上非视觉感官。

**不是公式**：不是"每N字加一次嗅觉"，而是问自己：这个场景的空气是什么味道？地板踩上去什么感觉？远处有什么声音？

| 感官 | 常见AI化写法（❌）                           | 人类化写法（✅）                                   |
| ---- | -------------------------------------------- | -------------------------------------------------- |
| 嗅觉 | "带着一股石屋特有的矿物质味道"（标签化说明） | 空气干得刮嗓子，每次呼吸都像在吞砂砾               |
| 听觉 | "轰鸣声"（概括）                             | 耳朵里只剩下一根尖细的嗡鸣，像红石火把快灭时的声音 |
| 触觉 | 几乎缺席                                     | 她把手按在台面上，木头被酒泡软了，指头陷进去一点   |
| 味觉 | 几乎缺席                                     | 嘴里全是铁锈味，不知道什么时候咬破了嘴唇           |
| 温度 | "干燥的近乎无菌的冷"（说明性）               | 冷气从脚底往上爬，像站在冰面上没穿靴子             |

**场景-感官映射示例**：

| 场景类型 | 应激活的感官                                                 |
| -------- | ------------------------------------------------------------ |
| 酒馆     | 酒液黏腻感、木桶闷响、发酵酸味混汗味、炉火的热               |
| 战场     | 耳膜嗡鸣、嘴里铁锈味、碎石敲头盔、硝烟呛鼻                   |
| 实验室   | 红石灯照久了眼睛干涩、石板划过手指的粗糙、红石火把的微弱嗡声 |
| 矿道     | 空气稀薄、脚下碎石滑动、远处水滴声、岩壁冰凉                 |

### 技术-情感打断规范

**核心问题**：AI倾向于用数据语言描写恐怖、用工程师视角描写恐惧。技术描写可以被角色使用，但必须被情感打断。

| AI化写法（❌）                                 | 人类化写法（✅）                                 |
| ---------------------------------------------- | ------------------------------------------------ |
| "步频恒定，步幅恒定，不受地形影响"             | 它走过来。每一步都一样。沙克斯的腿不听使唤       |
| "持续时间零点七秒，振幅四十七倍"               | 曲线猛地抬起。伊莱恩的手在抖，他读不准那个数字   |
| "波形包络线与泥板第三行符号的几何轮廓完全同构" | 他盯着那条曲线看了很久。然后他想起了橡木村的方向 |

**原则**：数据是角色看到的，但角色对数据的反应才是读者需要的。技术细节可以有，但要被人的反应打断——手抖、反复确认、突然想到别的事情。

## 严禁清单

### 绝对禁止的短语

- "眼中闪烁着XX的光芒"
- "心中涌起一股XX"
- "不禁XX"
- "一股XX感油然而生"
- "仿佛XX一般"（过度使用时）
- "宛如XX"（过度使用时）
- 任何网文常见套话

### 绝对禁止的写法

- 华丽排比句堆砌
- 为辞藻牺牲叙事完整性
- 角色大段内心独白解释主题
- 直接陈述"个体vs集体"主题
- 用环境描写代替情节推进（无目的的风景描写）
- 角色语音混淆（混混说话像学者）
- 回顾性措辞（"已经不知不觉""早已"等，违反现在时原则）
- 医学化身体描写（"瞳孔收缩""喉结滚动"等，违反展示而非告诉原则）
- 情感标签式身体描写（"嘴唇发抖""脊背僵直"等，直接告诉读者情感状态）
- 用比喻概括情感（"脑子里有什么碎了""像过载的机器"等——应改为行动的错位）
- 用技术语言描写恐怖（"步频恒定，步幅恒定"——应改为角色体验恐怖时的身体反应）
- 所有角色共享同一种内心独白风格（露利萨不应在心里做哲学分析）
- 金句均匀分布（金句应从情境中生长，密度不超过每2000字1个）
- 象征单一指向（核心意象每次都指向同一含义——应让象征变质）
- 「不是A，而是B」句式超过全文10次（AI最典型的修辞指纹）
- 「像」字明喻密度超过每1000字2个（"万物皆可比"是AI统计产物）
- 破折号（——）频率超过每1000字3个（AI用破折号模拟思维中断，用太多暴露机械性）
- 所有角色都说文学性语言（粗人必须说粗话，学者在压力下语言也必须退化）
- 一般叙事和对话中使用不必要的精确数值（"零点零七秒"→"不到一眨眼的工夫"）
- 每个章节/场景结尾都使用金句锤击模式（至少1/3用其他方式收束）
- 段落长度高度均匀（必须有剧烈波动：1-2行极短段和8行+长段都应存在）

### 绝对禁止的情节元素

- Minecraft中不存在的物品
- 现实世界科技
- 草药/中药体系
- 火药武器
- 破坏已建立设定的剧情

## 主题冲突表达指南

### 个体vs集体的隐性表达

**正确做法：**

- 角色在压力下做出选择，选择本身揭示立场
- 对话中透露优先级但不点名（"我的箭只够射一个人"）
- 言行矛盾（嘴上说为集体，行动在自保）
- 关键时刻的沉默（不表态本身就是表态）

**错误做法：**

- 任何角色直接讨论"个人与集体的关系"
- 旁白点明"他在个人利益和集体利益间挣扎"
- 用内心独白解释角色的哲学立场
