# 主题与布局系统

## 设计理念

主题由四个独立维度组成，每个维度可自由选择，任意组合：

- **背景风格** (8种) × **配色方案** (15种) × **字体方案** (8种) × **布局模板** (34种) = **5760种组合**

同一PPT的不同页面使用不同的布局模板，确保每页视觉框架不重复。

## 使用方式

```python
from scripts.spire.themes import combine, suggest_combination, generate_chapter_colors, ChapterColorStrategy
from scripts.spire.layouts import match_layout, ContentType, list_all_layouts
from scripts.spire.decorations import assign_decorations, render_decoration
from scripts.spire.shapes_animations import assign_animations

# 方式1：场景推荐
suggestion = suggest_combination("商务汇报")
theme = combine(**suggestion)

# 方式2：自由组合
theme = combine(
    background="赛博暗夜",
    color="霓虹科技",
    font="等宽极客",
    layout="左上标题+右下三卡片"
)

# 方式3：章节配色
colors = generate_chapter_colors(ChapterColorStrategy.PER_CHAPTER, "商务蓝", 4)

# 方式4：布局智能匹配
layout = match_layout(ContentType.CONTENT, used_layout_names=["左上标题右下三卡片"])

# 方式5：动画分配
animations = assign_animations(num_slides=9)
```

## 布局智能匹配

```python
from scripts.spire.layouts import match_layout, ContentType

# 根据内容类型和已用布局选取最佳布局
layout = match_layout(ContentType.CONTENT, used_layout_names=["左上标题右下三卡片", "底部标题上方双栏"])
```

匹配规则：

1. 优先选取同类型中未使用过的布局
2. 同类型布局耗尽时，回退到同类型已用布局
3. 同类型完全无匹配时，回退到所有未使用布局
4. 所有布局都用过时，随机选取

## 装饰元素系统

9种装饰类型，每页分配不同的装饰组合：

| 类型     | 说明                          | 变体            |
| -------- | ----------------------------- | --------------- |
| 渐变色带 | 顶部/底部/左侧/右侧的渐变色条 | 4个位置         |

| 水平线   | 横向分隔线                    | 实线/虚线       |
| 竖线     | 纵向装饰线                    | 实线/虚线       |
| 对角线   | 斜向装饰线                    | 虚线            |
| 圆点阵列 | 规则排列的小圆点              | 不同行列数      |
| 括号     | 左侧方括号装饰                | 不同高度        |
| 交叉线   | 十字虚线分隔                  | 居中            |
| 连接线   | 流程连接线/环形               | 水平/环形       |

```python
from scripts.spire.decorations import assign_decorations, render_decoration

# 分配装饰（自动去重）

```

## 章节配色策略

```python
from scripts.spire.themes import generate_chapter_colors, ChapterColorStrategy

# 统一配色：所有章节用同一方案
colors = generate_chapter_colors(ChapterColorStrategy.UNIFIED, "商务蓝", 4)

# 章节间变化：不同章节用不同配色（兼容背景适配）
colors = generate_chapter_colors(ChapterColorStrategy.PER_CHAPTER, "商务蓝", 4)
```

## 动画分配策略

```python
from scripts.spire.shapes_animations import assign_animations

# 为9页PPT分配动画（每页不同入场动画+切换效果）
animations = assign_animations(num_slides=9)

for i, (effect, transition) in enumerate(animations):
    engine.add_animation(slide, shape, effect)
    engine.set_transition(slide, transition)
```

规则：

- 页数≤16时，每页使用不同的入场动画效果
- 切换效果均匀分配，尽量不重复
- 使用固定种子确保可复现
